# Features
{: .no_toc }

An overview of everything taskpapr can do.
{: .fs-6 .fw-300 }

---

## Core features

### Kanban board
A classic column-based task board. Add columns, rename them, reorder them. Tasks live in columns and move between them as work progresses.

### Drag and drop
Move tasks between columns and reorder them within a column using drag and drop. Clean, fast, and reliable.

### Keyboard shortcuts
taskpapr is designed to be used without leaving the keyboard.

| Shortcut | Action |
|---|---|
| `N` | New task in focused column |
| `Enter` | Confirm / save |
| `Escape` | Cancel / close |
| `Ctrl+→` | Move card right |
| `Ctrl+←` | Move card left |

### Paper aesthetic
Warm cream backgrounds, calm typography, no visual noise. taskpapr looks good in daylight, at a coffee shop, or late at night.
